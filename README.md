# ALS Genetic Mechanism Explorer

Standalone static export of the Mustang Analytics ALS Genetic Mechanism Explorer.

This page presents a genetics-to-pathway-to-literature evidence graph for a 20-gene ALS panel. It integrates public identifiers and links from UniProt, STRING, ClinVar, Reactome, PubMed, and Open Targets, then renders ranked genes, mapped drugs, shared pathways, and machine-generated candidate mechanisms.

Important: the hypotheses are exploratory and machine-generated. They are not medical advice and are not validated biological mechanisms.

## Contents

- `index.html` — standalone project page with embedded hypothesis data.
- `site.css`, `motion.js`, images, and `fonts/` — only the presentation assets directly referenced by this project page.

Live page: https://mustanganalytics.com/projects/als-genetics-explorer/
